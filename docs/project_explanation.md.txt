# Project Explanation

## 📌 فكرة المشروع
المشروع عبارة عن نظام ذكي لترشيح الأفلام باستخدام الذكاء الاصطناعي.  
النظام يقدر يستقبل طلب المستخدم (زي: أفلام أكشن بتقييم عالي) ويرجع له أفلام مناسبة.

---

## 📊 معالجة البيانات (Data Preprocessing)
في البداية تم:
- تنظيف البيانات من القيم الناقصة
- تحويل الأعمدة الرقمية إلى أرقام
- تجهيز البيانات للاستخدام في الموديل

---

## 🤖 Machine Learning
تم استخدام أكثر من موديل:
- Linear Regression
- Random Forest (وكان الأفضل)

وتم تقييم الموديلات باستخدام:
- MSE
- R2 Score

---

## 🧠 معالجة النصوص (NLP)
تم تحويل بيانات الأفلام إلى نصوص، ثم:
- عمل Tokenization
- تنظيف النصوص

---

## 🔎 RAG System
تم استخدام embeddings لتحويل النصوص إلى أرقام ذكية  
وبعدها تم استخدام cosine similarity للبحث عن أفلام مشابهة لسؤال المستخدم

---

## 💬 توليد الرد (LLM Simulation)
النظام بيجمع الأفلام المناسبة ويعرضها للمستخدم في شكل رد منظم

---

## 🌐 واجهة المستخدم (Gradio)
تم إنشاء واجهة بسيطة تسمح للمستخدم بكتابة طلبه والحصول على توصيات

---

## 🎯 الخلاصة
المشروع بيجمع بين:
- Machine Learning
- NLP
- RAG
- Web Interface

وده بيخليه نظام متكامل قريب من الأنظمة الحقيقية
